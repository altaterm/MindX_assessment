# Task B: Fleet Arbitrage Dashboard

## Overview

Interactive React-based dashboard for visualizing maritime fleet compliance risk and simulating pooling strategies between surplus and deficit vessels.

## Features

### 1. Fleet Overview
- Total vessel count
- Surplus vessel count (compliant)
- Deficit vessel count (non-compliant)
- Real-time statistics

### 2. Liability Map
- Visual table/card layout of all vessels
- Color-coded compliance status:

- Key metrics per vessel

### 3. Pooling Simulator
- Interactive vessel selection
- Real-time compliance offsetting calculation
- Instant feedback on pooling effectiveness
